Multi-source domain adaptation (MSDA) traditionally assumes using all available subjects as sources improves performance.
However, Fig. 1 clearly shows why this fails:

Figure 1 illustrates:

  • Large domain gaps between many source subjects and the target.\
  • Combining all sources introduces noise and negative transfer.\

Many source subjects lie far from the target in feature space ‚Äî hurting adaptation.

‚úî Progressive Curriculum

Subjects are introduced one at a time (easy ‚Üí hard).

‚úî Density-Based Replay Memory

At each stage, representative samples are stored based on cluster density ‚Äî preventing forgetting.

‚úî Multi-domain Alignment

Using MMD-based discrepancy loss, the model aligns:

  • Replay samples\
  • Target domain

This ensures stable learning at every stage.
